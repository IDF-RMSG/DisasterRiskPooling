###README - Loss Simulator Data files

##Templates

- LossSimulator_uploadTemplate.xlsx: Source file for historical data upload, for when user wants to import a historical record of event losses (different to EM-DAT default data) to feed through the Loss Simulator. Uses can download this template on the '91Data Selection'92 page, add their data and upload that data in the same page.

- LossSimulatorOutput.csv: Example output file that can be downloaded after the Loss Simulator has been run. The download button creates a new file each time; this template is for information but is not used directly in the creation of download files.
