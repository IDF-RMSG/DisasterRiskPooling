##Loss Simulator

This folder contains code for an R-Shiny interface, as part of the Disaster Risk Pooling Tool, which enables users to pull data from online historical disaster loss catalogues, trend them according to changes in population, inflation and GDP, then export modelled loss curves for use in the Risk Pool Structuring tool (see folder 'RiskPoolStructuring'). This app is based on an original tool by the World Bank DFRIP.

The Loss Simulator is available online at: https://idf-rmsg.shinyapps.io/DisasterRiskPooling
